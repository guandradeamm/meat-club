> A simple react.js and tailwind CSS website for selling meat to your customers.

---

**Technologies used:**

- React.js
- Tailwind CSS

---

**Configuration:**

1. After cloning the respository, install the dependencies:

- using npm:

```
npm install
```

- using yarn:

```
yarn
```

2. To run the application locally you can utilize the start script:

- using npm:

```
npm run start
```

- using yarn:

```
yarn start
```
